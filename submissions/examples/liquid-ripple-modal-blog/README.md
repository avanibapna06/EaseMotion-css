# Liquid Ripple Modal (#38461)

## Description
A CSS-only modal that utilizes a "Liquid Ripple" scale transition. By morphing `border-radius` and `transform: scale`, the modal expands like a fluid bubble from the center.

## Key Features
- **Fluid Motion:** Uses a specific `cubic-bezier` timing for a "liquid" feel.
- **Pure CSS:** No JavaScript required; uses the checkbox state hack.
- **Responsive:** Adapts to different screen sizes smoothly.

## Usage
Add the `modal-wrapper` to your blog layout. The checkbox state manages the visibility and entry transition.
