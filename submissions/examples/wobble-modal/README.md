# Wobble-Focus Modal Component (#34077)

## Description
A pure CSS modal component designed for creative portfolio layouts. It features a "Wobble-Focus" entrance animation using custom cubic-bezier timing, providing a high-end interactive feel without any JavaScript dependencies.

## Key Features
- **Performance:** Optimized CSS keyframes for 60fps animations.
- **Responsive Design:** Adapts fluidly to different screen sizes.
- **Zero Dependencies:** Fully functional with vanilla HTML/CSS.

## Usage
Add the `modal-overlay` and `modal-box` structure to your HTML to implement the component. The animation will trigger automatically upon rendering.

## Customization
- You can modify the `animation-duration` or the `cubic-bezier` values in `style.css` to adjust the "wobble" intensity.
