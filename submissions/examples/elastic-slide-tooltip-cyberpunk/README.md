# Elastic Slide Tooltip (#38505)

## Description
A modern, Cyberpunk-inspired tooltip component featuring a smooth elastic slide animation using `cubic-bezier` timing.

## Key Features
- **Elastic Motion:** Uses a custom `cubic-bezier` for the "spring" effect.
- **Neon Aesthetic:** High-contrast cybernetic styling.
- **Zero JS:** Fully functional using pure CSS hover states.

## Usage
Simply wrap your trigger element in the `.tooltip-container` and add the `.tooltip` structure as shown in `demo.html`.
